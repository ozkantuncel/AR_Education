# Titanic AR Education

## Project Overview
**Titanic AR Education for Kids** is an Augmented Reality application designed to make learning and playtime more engaging and interactive for children. This project was developed as part of the **Software Engineering Current Topics** course.

## Key Features
- Fun and educational AR games tailored for children.
- Safe, colorful, and intuitive design to enhance kids' creativity and learning.
