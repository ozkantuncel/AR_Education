# Titanic AR Education for Kids

**Titanic AR Education for Kids** is an engaging and interactive Augmented Reality (AR) application designed specifically for children. This project aims to make learning fun and interactive by blending education with immersive AR experiences. Developed as part of the **Software Engineering Current Topics** course, this project brings history to life through a creative and playful AR-based approach.
